# Tech Stack

Technologies used in the Workflow Builder.

## Frontend

- **Next.js 14** - React framework with App Router
- **React 18** - UI library
- **Tamagui** - UI component library
- **React Flow** - Visual workflow canvas
- **tRPC Client** - Type-safe API calls
- **React Query** - Data fetching and caching

## Backend

- **Next.js API Routes** - Server-side endpoints
- **tRPC 10** - Type-safe RPC framework
- **Zod** - Runtime type validation
- **SuperJSON** - Serialization for complex types

## Database

- **Supabase** - PostgreSQL + Auth + RLS
- **PostgreSQL 15** - Relational database
- **Row-Level Security** - Multi-tenant security

## Workflow Engine

- **Temporal** - Workflow orchestration
- **@temporalio/client** - Temporal client
- **@temporalio/worker** - Temporal worker
- **@temporalio/workflow** - Workflow definitions

## Development Tools

- **TypeScript** - Type safety
- **Vitest** - Unit testing
- **Playwright** - E2E testing
- **ESLint** - Code linting
