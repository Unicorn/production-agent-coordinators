2. **Set up local development** - See [Local Development](local-development.md)

## For Developers

If you're setting up a development environment:

1. Review [Installation](installation.md) for prerequisites
2. Follow [Local Development](local-development.md) for complete setup
3. Check [Development Guide](../development/README.md) for contributing guidelines

## What You'll Need

Before starting, ensure you have:

- **Node.js** >= 18
- **Yarn** >= 1.22
- **Supabase** account (or local Supabase CLI)
- **Temporal** dev server (for workflow execution)
